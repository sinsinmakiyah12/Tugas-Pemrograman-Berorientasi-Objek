Aplikasi MindBuddy ini memenuhi kriteria tugas Pemrograman Berorientasi Objek sebagai berikut:
1. Berbasis Mobile: Saya buat menggunakan Android Studio dan sudah bisa diinstal di Android.
2. Penggunaan CRUD: Saya menerapkan fungsi create, read, update, dan delete secara lengkap untuk pengelolaan data jurnal pengguna.
3. Implementasi 4 Pilar OOP
   *Encapsulation: Di file Journal.java, data jurnal (judul, isi) disimpan dalam variabel private. Akses ke data tersebut dilakukan melalui metode public yaitu getter dan setter.
   * Inheritance (pewarisan): Di file MainActivity.java, kelas ini menggunakan kata kunci extends.
   * Polymorpishm: Di file MainActivity.java pada metode @override protected void onCreate untuk memunculkan tampilan jurnal.
   * Abstraction: Di file DatabaseHelper.java, pengguna hanya perlu menekan tombol simpan dilayar tanpa tahu proses nya.
4. Koneksi Database: Aplikasi ini terkoneksi dengan SQLite sebagai database lokal untuk penyimpanan data yang permanen dan aman.
